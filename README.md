# DemoApp

A sample android app that shows how to use coroutines, hilt-dagger and mvi in Kotlin by Clean Architecture.

# Built With
* Kotlin
* MVI
* Clean Architecture
* Compose
* Work manager

# Libraries
* Coroutines
* Dagger (Hilt)
* Retrofit
* gson
* okhttp
* Timber
* MockWebServer
* Work manager
